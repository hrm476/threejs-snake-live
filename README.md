# SNake 3D

This repo is a 3D Snake game built with Three.js and GSAP, featuring multiple color palettes, audio controls, and a smooth orbit-style camera experience.

[Play the game](https://hrm476.github.io/threejs-snake-live/)

![Screenshot 2023-10-11 alle 16 47 55](https://hrm476.github.io/threejs-snake-live/og-preview.png)

## Installation

Install the dependencies

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
